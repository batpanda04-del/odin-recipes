README

Recipes is a simple HTML program for The Odin Project

Skills the project will demonstrate:
    
    - Show basic understanding and use of Git repositories with SSH keys
    - Write useful commit messages to Git with proper syntax
    - Commit iterations periodically and with intention when big changes to code are made
    - Write a functional HTML boilerplate and know how each element works with the Web Browser
    - Use common HTML tags with basic web accessibility and security in mind
    - Properly display images and insert links
    - Get hungry talking about and looking at food
    - Have fun, learn good
